Leviattan utils, layer 1

## Installation

Install this via pip (or your favourite package manager):

```bash
pip install lvtn_utils
```

## Development

Install locally into virtualenv

```bash
virtualenv .venv
source .venv/bin/activate
pip install .[dev]
pip install .
pre-commit install
pre-commit install --hook-type commit-msg
```
