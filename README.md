Leviattan utils, layer 1

## Installation

Install this via pip (or your favourite package manager):

```bash
pip install lvtn1-utils
```

## Development

Install locally into virtualenv

```bash
virtualenv .venv
source .venv/bin/activate
python rtool.py install
```
